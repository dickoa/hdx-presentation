What is HDX?

L'Humanitarian Data Exchange (HDX) est une plateforme ouverte de OCHA pour l'échange de donnée humanitaire. L'objectif de HDX est rendre les données humanitaires facile à trouver and facile à utiliser pour des analyses.

Qu'entends on par données humanitaire?

1. Données de référence

2. Données de réference (like CODs)

3. Données sur la situation (besoin ou evaluation des dommages, logistique)

4. Données de reponses (3W, etc.)
