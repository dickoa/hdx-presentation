C'est quoi HDX?

L'Humanitarian Data Exchange (HDX) est une plateforme ouverte de OCHA pour l'échange de donnée humanitaire. L'objectif de HDX est de rendre les données humanitaires facile à trouver and facile à utiliser pour des analyses.

C'est quoi les données humanitaires?

1. Données de référence

2. Données de réference (like CODs)

3. Données sur la situation (besoin ou evaluation des dommages, logistique)

4. Données de reponses (3W, etc.)

Et les metadonnées? les données sur les données.

Pourquoi les metadonnées sont elles importantes?

  • Recherche: des bons tags et descriptions rendent vos données plus facilement découvrable
  • Confiance: Nos utilisateurs nous ont dit à plusieurs reprise combien il est important de savoir à quel point ils peuvent avoir confiance aux données.
  • Fitness for use: Les méthodes utilisées pour collecter ou traiter les données peuvent influencer (suitability) de l'information pour différent objectif: la methodologie!
  • Utilisabilité: Sans bonne documentation des licences, méthodes et nom de champs, les utilisateurs peuvent ne pas pouvoir utiliser les données que vous avez partagées.
